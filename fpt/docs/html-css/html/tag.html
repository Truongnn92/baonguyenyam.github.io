4) Tag Type

Type TAGS Wrong
Tag về định dạng văn bản

Tag mềm: span, a, small, strong, b, u, i, sub, sup...
Tag cứng: div, p, h1, h2, h3, h4, h5, h6, hr...

Trong đó

  • Tag div: có độ ưu tiên cao nhất
  • Tag h1 > h6, hr: ko nên có tag con
  • Tag p: nên có tag con là tag mềm

	*

[/p]
